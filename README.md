# Data Cleaning & Visualization Project

## Project Overview ::
This project demonstrates data cleaning, preprocessing, and visualization techniques using Python. A raw sales dataset was analyzed to identify missing values, remove duplicate records, and generate meaningful visual insights.

## Objectives ::
* Handle missing values
* Remove duplicate records
* Process and clean raw data
* Visualize trends and relationships
* Create insightful reports using graphs

Technologies Used ::
* Python
* Pandas
* Matplotlib
* Seaborn

## Data Cleaning Steps ::

1 Missing Values 
* Numerical columns were filled using mean values.
* Categorical columns were filled using mode values.

2 Duplicate Records
* Duplicate rows were identified and removed.

3 Dataset Export
* Cleaned data was saved as `cleaned_data.csv`.

## Visualizations Created ::
 1. Distribution Plot
Shows the distribution of numerical data.

 2. Scatter Plot
Displays the relationship between two numerical variables.

 3. Correlation Heatmap
Highlights correlations between numerical features.

## Output Files ::
* cleaned_data.csv
* distribution.png
* scatter_plot.png
* heatmap.png
